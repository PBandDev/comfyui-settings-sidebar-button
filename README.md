# ComfyUI Settings Sidebar Button

Adds a button to the sidebar to instantly open the Settings dialog panel. Saves you a click!

<img src="assets/preview.gif" alt="User clicking on a settings button which opens the settings dialog window" loop=infinite>

---

## Installation

1. Open **ComfyUI**.
2. Go to **Manager > Custom Node Manager**.
3. Search for `Settings Sidebar Button`.
4. Click **Install**
